Overview

This practical walks through three established methods for computing polygenic risk scores (PRS) from GWAS summary statistics, without requiring any individual-level GWAS data. Each method is evaluated in a multi-ancestry validation cohort (1000 Genomes Project, N = 2,504) across five super-populations.

1

C+PT — Clumping + P-value Thresholding

Select independent GWAS SNPs by LD clumping, then tune the P-value inclusion threshold using a validation sample to maximise predictive accuracy.

PLINK 1.9 + R
2

SBLUP — Summary-data BLUP

Ridge regression on GWAS summary statistics using an LD correlation matrix to compute joint SNP effects across all M variants simultaneously.

R + optional GCTA
3

SBayes C — Bayesian Sparse Regression

A two-component spike-and-slab prior estimated via MCMC. More flexible than SBLUP — estimates the proportion of causal SNPs (π) from the data.

GCTB
Learning Objectives
🧬
Understand how marginal GWAS effects differ from joint effects due to LD
🔧
Apply C+PT, SBLUP, and SBayes C using PLINK, R, and GCTB
📊
Evaluate and compare PRS accuracy across five ancestry groups
🌍
Interpret the role of the LD reference panel and its ancestry limitations
Quick Start
  1. Clone the repository and install tools
    bashTerminal
    git clone https://github.com/moeenriaz2/SISG26-PRS.git
cd SISG26-PRS
bash setup.sh              # installs PLINK 1.9, PLINK 2, GCTB → exe/
export PATH="$PWD/exe:$PATH"
  2. Create working directory
    bashTerminal
    mkdir -p practical/{C+PT,SBLUP,SBC}
cd practical
  3. Download 1000 Genomes genotype data (~20 min)
    bashTerminal — run from repo root
    bash data/download_1kg.sh
# Downloads chr20 VCF from EBI FTP, converts to PLINK format (--double-id, --set-all-var-ids)
# Creates: 1kg_hm3.bed/.bim/.fam  +  EUR/EAS/SAS/AMR/AFR.id
  4. Download GWAS summary stats, phenotypes, and LD matrices
    bashTerminal
    cd practical
wget https://github.com/moeenriaz2/SISG26-PRS/releases/download/v1.0/gwas_summary_stats.tar.gz
wget https://github.com/moeenriaz2/SISG26-PRS/releases/download/v1.0/phenotypes.tar.gz
wget https://github.com/moeenriaz2/SISG26-PRS/releases/download/v1.0/ldm_chr20.tar.gz
tar -xvzf gwas_summary_stats.tar.gz
tar -xvzf phenotypes.tar.gz
tar -xvzf ldm_chr20.tar.gz
  5. Install R packages
    RR Console
    install.packages(c("data.table", "ggplot2", "dplyr", "tidyr"))
  6. Open the practical and follow along

    → practical.html — step-by-step instructions for all three methods with code, questions, and discussion.
